Summary

Created a GitHub page to showcase class projects and professional experiences. The page is divided into About Me, Education, Course, Project, and Skill sections to give users an idea of accomplishments.

Goals

  • Showcase projects that I worked on while in college.
  • Show experiences to serve as an online resume.

Constraints

Project is hosted on GitHub Pages which have the following usage limits:

  • Sites must be no larger than 1 GB in size.
  • Soft Bandwidth limit of 100 GB per month.
  • Soft Bandwidth limit of 10 builds per hour.

Technical Constraints

  • Not designed to handle a large amount of web traffic.
  • Site data is hosted in the GitHub repository.

Challenges

  • Using CSS with Bootstrap classes for the first time.
  • Getting used to the Bootstrap framework for the first time.

Lessons Learned

  • Limited to how many projects I could add without negatively impacting the user experience. It may take longer for a user to find a specific project if multiple projects are on the same page.
  • The skills section using number ranges can be misleading. The section could confuse users and not be straightforward.
